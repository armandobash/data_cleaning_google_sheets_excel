# Data Cleaning in Google Sheets & Excel
This exercise help display my abilities in cleaning databases in excel or google sheets.

This repository shows how I structure, clean, and transform data to make it ready for analysis and tracking using Excel and Google Sheets.

What I cover:

- Data cleaning: remove duplicates, fix blanks, trim extra spaces, standardize text/casing
- Data transformation: split/merge columns, format dates, convert data types, create calculated fields
- IDs & timestamps: generate record IDs (e.g., TXN_#######) and add fixed “created at” timestamps
- Validation & consistency: dropdowns, data validation rules, error checks, and standard formats
- Tracking-ready structure: set up tables for clean data entry (headers, required fields, status tracking)
- Lookup mapping: connect labels to values (e.g., Coffee → 2) using lookup tables

Goal: turn messy data into a clean, consistent dataset that’s easy to enter, track, and analyze.
